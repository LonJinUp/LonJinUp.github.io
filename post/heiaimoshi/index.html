两行代码为网页添加暗黑模式

2019-04-11

自从微信支持暗黑模式以后，很多网站也加了暗黑模式，这个可以利用css的filter即可做到。直接看代码吧！

实现代码

  html{
      filter: invert(1) hue-rotate(180deg);
  }
<!-- 逆转图片颜色 -->
  html img{
      filter: invert(1) hue-rotate(180deg);
  }

  • 如果想深入了解的，可以看一下相关文档 filter详解

  • 基本代码就这么两条，大家可以试一下效果。
