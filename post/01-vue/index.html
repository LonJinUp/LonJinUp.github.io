01-Vue基础知识

2019-08-05 10 min read # VUE

Vue

虽然年后就说开始学vue了，但一直断断续续的，得系统性学习一波了，笔记也顺便记录一下，方便查阅。

基础笔记基本来源于官方文档。

vue起步

  • 初学可以先用script引入，先过一遍官方文档
<script src="https://cdn.jsdelivr.net/npm/vue"></script>

模板语法

  • 现在开始，创建一个vue实例。
<body>
   <div id="app">{{message}}</div>
</body>
<script src="https://cdn.jsdelivr.net/npm/vue"></script>
<script>
    var app=new Vue({
        el:"#app", //指定绑定的目标dom元素的Id
        //绑定数据源
        data:{
            message:"Hello World By Vue"
        }
    });
</script>

  • 这样，一个vue应用已经创建完成，可以看到，首选new了一个vue对象，然后通过el去绑定元素，通过在data定义一个值，然后通过类似模板语法的方式，输出到页面。此时页面属于响应式的，可以通过在控制台修改msg来看是否改变(app.message="hello World")。

  • 很多场景下不需要去根据数据改变而更新页面数据，此时可以用v-once

<body>
   <div id="app" v-once>这个数据不会变{{message}}</div>
</body>
<script src="https://cdn.jsdelivr.net/npm/vue"></script>
<script>
    var app=new Vue({
        el:"#app", //指定绑定的目标dom元素的Id
        //绑定数据源
        data:{
            message:"Hello World By Vue"
        }
    });
</script>

指令

v-html

  • 双大括号会将数据解释为普通文本，而非 HTML 代码。为了输出真正的 HTML，你需要使用 v-html 指令：
<body>
   <div id="app">
       <div v-html='rawhtml'></div>
   </div>
</body>
<script src="https://cdn.jsdelivr.net/npm/vue"></script>
<script>
    var app=new Vue({
        el:"#app", //指定绑定的目标dom元素的Id
        //绑定数据源
        data:{
            rawhtml:"<p>我要输出html<p>"
        }
    });
</script>

v-bind

  • 还可以通过v-bind绑定元素特性,v-bind在vue中称为指令。
<body>
    <div id="app2" v-bind:title="message">鼠标放在这试一试</div>
</body>
<script src="https://cdn.jsdelivr.net/npm/vue"></script>
<script>
    var app=new Vue({
        el:"#app2", //指定绑定的目标dom元素的Id
        //绑定数据源
        data:{
            message:'页面加载于'+new Date().toLocaleDateString(),
        }
    });
</script>
  • Vue中还可以把bind指令缩写
<body>
    <!-- 完整语法 -->
    <div id="app2" v-bind:title="message">鼠标放在这试一试</div>
    <!
    -- 缩写 -->
    <div id="app2" :title="message">鼠标放在这试一试</div>
</body>

v-if 与 v-else v-else-if

  • 很多场景下需要控制标签的显示隐藏，可以通过Vue中提供的v-if.
<body>
    <div id="app3">
        <p v-if="show">你现在可以看到我</p>
        <p v-else>这是else</p>
    </div>
</body>
<script src="https://cdn.jsdelivr.net/npm/vue"></script>
<script>
    var app3=new Vue({
        el:"#app3", //指定绑定的目标dom元素的Id
        //绑定数据源
        data:{
            show:true
        }
    });
</script>
  • 此时data中show:true是显示的，我们在控制台或者代码中改成show:false,可以看到，标签不显示了。
  • v-else必须紧跟在v-if或者v-else-if后面。指令必须绑定到一个标签上去，可以绑定到 <template> 不可见的元素上去，方便控制。

v-show

  • 我们还可以通过v-show来控制元素的显示与隐藏
<body>
    <div id="app4">
        <p v-show="show">你现在可以看到我</p>
    </div>
</body>
<script src="https://cdn.jsdelivr.net/npm/vue"></script>
<script>
    var app4=new Vue({
        el:"#app3", //指定绑定的目标dom元素的Id
        //绑定数据源
        data:{
            show:true
        }
    });
</script>
  • 我们发现v-show指令中，当data中show:false时候，元素行内样式中是display：none

  • 由此可以说v-if才是真正的条件渲染，当条件不满足时候，标签并不会去渲染，直至满足条件，而v-show是先把元素渲染出来，如果不满足条件，则dispaly=none

  • 一般来说，v-if 有更高的切换开销，而 v-show 有更高的初始渲染开销。因此，如果需要非常频繁地切换，则使用 v-show 较好；如果在运行时条件很少改变，则使用 v-if 较好。

v-for

  • Vue还提供了列表循环指令
<body>
    <div id="app5">
        <ul v-for="item in list">
            <li v-bind:data-id="item.id">{{item.name}}</li>
        </ul>
    </div>
</body>
<script src="https://cdn.jsdelivr.net/npm/vue"></script>
<script>
     var app5=new Vue({
        el:"#app5",
        data:{
            list:[
                {id:1,name:"js"},
                {id:2,name:"jquery"},
                {id:3,name:"vue"},
            ]
        }
    })
</script>
  • 我们可以在控制台输入app5.list.push({id:4,name:"zepto"})可以看到，列表追加了一行。同时通过v-bind给标签加了一个data-id属性

v-on

  • 为了让用户和你的应用进行交互，我们可以用 v-on 指令添加一个事件监听器，通过它调用在 Vue 实例中定义的方法：
<body>
    <div id="app6">
      <button v-on:click="ClickMe">{{message}}</button>
    </div>
</body>
<script src="https://cdn.jsdelivr.net/npm/vue"></script>
<script>
    var app6=new Vue({
        el:"#app6",
        data:{
            message:"请点击我"
        },
        methods:{
            ClickMe:function(){
                this.message="狗子，你变了！";
                console.log(1)
            }
        }
    });
</script>
  • 在修改标签内容时候，我们并没有像jQuery一样去操作DMO,所有的 DOM 操作都由 Vue 来处理。同时v-on也可以进行简写
<body>
    <div id="app6">
       <!-- 完整语法 -->
      <button v-on:click="ClickMe">{{message}}</button>
      <!-- 缩写 -->
      <button @click="ClickMe">{{message}}</button>
    </div>
</body>

v-model

  • Vue 还提供了 v-model 指令，它能轻松实现表单输入和应用状态之间的双向绑定。
<body>
    <div id="app7">
        <p>{{message}}</p>
        <input type="text" v-model="message">
    </div>
</body>
<script src="https://cdn.jsdelivr.net/npm/vue"></script>
<script>
    var app7=new Vue({
       el:"#app7",
       data:{
           message:"hellow"
       }
   })
</script>

组件

  • 组件系统是 Vue 的另一个重要概念，因为它是一种抽象，允许我们使用小型、独立和通常可复用的组件构建大型应用。我们所写的页面都可以分割成一个个组件，比如头部，轮播，尾部等...

创建组件

  • 在Vue中，创建组件就是预定义了一个Vue的实例。
    <script>
    //创建一个名字为todo-item的组件
    Vue.component('todo-item',{
        template:"<li>这是一个实例</li>"
    });
    </script>
  • 如果这样使用只能渲染相同文本，所以可以通过定义一个prop可以接收从父组件传过来的数据
    <div id="app">
        <ol>
            <todo-item v-for='item in list' v-bind:todo='item' v-bind:key='item.id'>

            </todo-item>
        </ol>
    </div>
    <script>
        //定义组件
        Vue.component('todo-item',{
            props:['todo'],
            template:"<li>{{todo.text}}</li>"
        });

        var app=new Vue({
            el:"#app",
            data:{
                list:[
                    {id:1,text:"hellow world"},
                    {id:2,text:"hellow"},
                    {id:3,text:"hi"},
                    {id:4,text:"world"},
                ]
            }
        })
    </script>
